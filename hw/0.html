HW 0 - Classes

Create a class Student whose constructor takes the following parameters:
  • Name of student
  • Age of student
  • Home country
  • Student's Major
  • A dictionary which maps his courses to grades

    • The class should have the following methods:
  • calculateGPA()
    which returns his gpa out of 4
    Use scheme:- A: 4.0, B: 3.0, C: 2.0, F: 1.0
    Assume all courses have equal credits
  • addCourse(CourseName, Grade)
    add course with grade to student object.

    • Printing a student object on console should print in the following format:
    <Student Name>, <Age>, <Home Country>
    Major: <Major>
    GPA: <GPA>

    Example:

    >>> student1 = Student('John Petrucci', 23, 'USA', 'Computer Science' {'Python Programming':'A'})
    >>> print student1
    John Petrucci, 23, USA
    Major: Computer Science
    GPA: 4.0
    >>> student1.addCourse('Data Structures', 'B')
    >>> print student1
    John Petrucci, 23, USA
    Major: Computer Science
    GPA: 3.5
